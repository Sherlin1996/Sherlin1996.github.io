About Me

[ 前言 ]
  「態度決定高度」這是我的父母從小教導我的人生態度。人生道路上並非總是一路順風，但態度會決定結果，是父母的教育使我在面對挫敗時，拿出積極的態度面對，正視問題且相信自己。

[ 個人經歷 ]
  我是徐瑋羚，個性樂觀，處事風格細心謹慎，因此在團隊合作中能與同伴維持良好關係並同時能達成工作目標。在求學期間多次擔任組長負責規劃專案分工、時間控管..等，透過與組員互相溝通協調，也培養組織應對的能力。擔任組員時也常被指派上台報告的角色，透過在校報告的經驗的累積，也幫助我提升口條及應對的能力。畢業後我並沒有選擇報考公務員，作為我往後的道路，而是相信自己的個性更適合快速變動的領域。

[ 轉職動機&學習過程 ]
  在職場一段時間後，開始反思自己所學的技能與生涯規劃，想趁年輕盡可能多接觸不同領域增廣見聞。隨著科技進步越來越快，掌握多種軟硬實力並保持不斷學習，才能應變市場變化。原工作到一段落後，我選擇離職，從自己最有興趣的「數據」開始探索。在實習期間，從一開始無程式經驗到踏出舒適圈，在過程中，透過研究程式、技術交流，使我更喜歡程式語言帶來的成就感。在參與產品開發中，跨部門與前端工程師合作，接觸前、後端API串接與資料庫相關知識，運用Python、MySQL、AWS，進一步研究Python RESTful API 技術，藉此經驗，我更加了解前後端的互動與架構、跨部門溝通的重要性。

[ 未來展望 ]
  處在快速發展的世代，我時時刻刻提醒自己，抱持求知的態度並保持謙虛，去面對動盪的未來。我相信我的能力及經驗能夠使我快速掌握職務內容、發揮所長，為團隊達成目標，期待能夠進入貴公司，與優秀的團隊共創未來。
